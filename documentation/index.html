Structure of CSS/JS files

In 'css' folder contains used template styles and styles of 'jQuery' plugins.

  • assets/css/bootstrap.min.css Bootstrap, mobile first front-end framework
  • assets/css/animate.css Just-add-water CSS animations
  • assets/css/jquery-ui.css jQuery user interface interactions
  • assets/css/font-awesome.min.css Font Awesome, scalable vector icons
  • assets/css/bundle.css Default css
  • assets/css/meanmenu.min.css For mean menu
  • assets/css/normalize.css For Normalize css
  • assets/css/owl.carousel.css Core Owl Carousel CSS File
  • assets/css/owl.video.play.css Owl Carousel video
  • assets/css/magnific-popup.css magnific-popup
  • assets/css/pe-icon-7-stroke.css pe-icon-7-stroke css
  • assets/css/demo.css offcanvas css
  • assets/css/menu_elastic.css offcanvas menu css
  • assets/css/style.css Template styles
  • assets/css/responsive.css Responsive styles

JavaScripts are located in 'js' folder

  • assets/js/vendor/jquery-1.12.0.min.js jQuery, JavaScript library
  • assets/js/vendor/modernizr-2.8.3.min.js Modernizr
  • assets/js/jquery.meanmenu.js jQuery meanMenu
  • assets/js/jQuery.imagesloaded.pkgd.min.js imagesloaded js
  • assets/js/jquery.magnific-popup.min.js magnific-popup
  • assets/js/isotope.pkgd.min.js isotope.pkgd.min
  • assets/js/jquery.validate.min.js jquery.validate.min
  • assets/js/snap.svg-min.js snap.svg-min.js
  • assets/js/classie.js classie scripts
  • assets/js/main3.js offcanvas scripts
  • assets/js/main.js Template scripts
  • assets/js/plugins.js Init jQuery scripts
